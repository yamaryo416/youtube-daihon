■ 本編 1: Claude in Chrome — 情報収集の「往復作業」が完全になくなる自動操作機能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一つ目は、Claude in Chrome。

Claude in Chrome は、Google Chrome の拡張機能として提供されている、ブラウザ自動操作機能です。

「Chrome の拡張機能？なんか難しそう...」そう思いましたよね。

全然難しくないんです。

インストールはたったの 2 ステップ。
Google Chrome を開いて、

「Claude in Chrome」と検索して、

Claude - Chrome ウェブストアをクリック

このようなページが開きますので、「Chrome に追加」をクリック。

このようなモーダルが表示されますので、「拡張機能を追加」をクリックしましょう。

Claude のログイン画面が表示されますので、ログインしましょう。

ここで一つ、お伝えしておきます。
Claude in Chrome は有料プラン（月額 3,000 円ほど）が必要です。
ただ、これから見せる機能を見てもらえれば、この金額は安すぎると感じると思います。
まだ無料プランの方は、ぜひこれを機に試してみてください。

ログインすると、ブラウザの右上に Claude のオレンジ色のアイコンが表示されます。
クリックするだけで、どのページを開いていても、すぐに Claude を呼び出せます。

で、何がすごいかというと。

「Google カレンダーに予定を入れたい」とか「申し込みフォームに情報を入力して送信したい」というとき、今まで何をやっていたか、振り返ってみてください。

自分でアプリやフォームを開いて、必要な情報を一つひとつ手入力して、送信・保存して...。

この一連の操作を、Claude in Chrome は全部やってくれます。

試しに「明日の 10 時に taro さんとオンライン MTG の予定を Google カレンダーに入れて、MTG の URL を教えてください」と指示してみましょう。

Claude のプランというものが出力されましたね。

具体的に何をするか書いてありますね。

これでプランを承認をクリックしましょう。

自動で、Google Calendar が開きました。

見てください！矢印が表示されて、ブラウザを操作していますね！

気づけば、指示通りにカレンダーに予定が作成されて、右側に MTG の URL が出力されました！

すごすぎませんか！
僕自身、これを初めて使った時、え、めっちゃ楽じゃんで、感激しました！

これ、ブラウザで開いているページならどんなページにでも使えるので、Web サイトやアプリを作った後の確認作業にも応用できます。

今回は、試しで作ったタスク管理アプリを開いて、動作確認をしてもらいます。
皆さんも自分で作ったサイトやアプリがあれば、ぜひ一緒に開いてみてください。
デザインの崩れ、リンク切れ、文章のおかしな点、動作のバグ——気になるところを Claude に投げるだけで、その場でチェックしてもらえます。
こう指示するだけです。

━━━━━━━━━━━━━━━━━
【指示】
このタスク管理アプリの動作を確認してください。
・タスクの作成・編集ができるか
・タスクを完了・未完了に切り替えられるか
・完了・未完了のタスク数が正しく表示されるか
問題があれば教えてください。
━━━━━━━━━━━━━━━━━

Claude がアプリ全体を確認して、「タスクを完了にしても未完了の件数が更新されていません」「編集後に保存ボタンを押しても反映されないケースがあります」と、問題点を具体的に教えてくれます。

作ったアプリの動作チェックを、自分で一つひとつ試さなくても、Claude に丸投げできる。
これが地味に、めちゃくちゃ助かるんですよ。

また、ページの翻訳・要約にも便利です。

Web サイトやアプリ作成を本格的に進めていくと、英語のドキュメントや技術的な説明ページを読まなければいけない場面が出てきます。
そういうとき、英語を翻訳したり用語を調べるのに、別でチャット AI を開いて内容をコピペして...という手間が、全部なくなります。
開いているページをそのまま Claude に読ませて「このページを日本語で要約して」「この部分をわかりやすく解説して」と聞くだけ。
調べながら手を動かす流れが、一切止まらなくなるんですよ。

■ 本編 2: Cowork — PC を丸ごと動かせる「AI の同僚」がデスクトップに現れた

神機能 2 つ目は、Cowork です。

Claude のデスクトップアプリに搭載された機能で、PC 自動操作機能です。

Claude in Chrome は「ブラウザの中だけ」を動かしますが、Cowork は「PC そのもの」を動かします。
フォルダの整理、ファイルの作成・保存、アプリの操作——こういった PC 全体に関わる作業を、全部 Claude に任せられます。

Cowork は Claude のデスクトップアプリから使います。

Claude のデスクトップアプリをまだインストールしていない方は、
Web 版の Claude から、左下をクリック
アプリと拡張機能を入手をクリック
デスクトップの macOS 版をダウンロードをクリックすることでダウンロードできます。

ダウンロードが完了したら、Claude のアプリを開きましょう。
このような画面が表示されますので、
左上のタブの Cowork をクリックすることで、Cowork を使うことができます。

モデルは Opus を選ぶのがおすすめです。
一番賢いモデルを選ぶことで、複雑な指示でもスムーズに動いてくれます。

では早速、こう指示してみます。

━━━━━━━━━━━━━━━━━
【指示】
明日の 11 時に taro さんとオンライン MTG の予定を Google カレンダーに入れて、
デスクトップにアジェンダファイルを作成してください。
また、メールアプリを開いて、 MTG の招待メールを良い感じに書いて、送れる状態にしてください。

不足している情報を Claude が聞いてくれます。

この場所で作業していいか、聞かれるので、許可をクリック

このタスクを実行して良いか聞かれますが、これは、Google Calendar のアプリでこの内容で
予定を登録して良いか？ということなので、このタスクに許可をクリック

このタスクを実行して良いか聞かれますが、これは、Gmail アプリでにこの内容で、
下書きのメールを作成して良いか？ということなので、このタスクに許可をクリック

「AI が勝手に PC を操作するって、なんか怖くないですか？」そう思いましたよね。
でも Cowork は、一つひとつのタスクを実行する前に必ず「やっていいですか？」と確認を取ってきます。
何をやるか自分で把握しながら進められるので、思いがけない操作をされる心配はありません。

1 分ほどで完了しました！

どうですか？
カレンダーを見ると、予定が入っており、
デスクトップに、デスクトップに「MTG_taro_20260530.txt」というファイルを自動で作成されていて、
メールアプリを起動して、ミーティング URL と丁寧な招待文を本文に入力した状態で止まってくれます。

これ、やばすぎませんか？

僕も最近、めちゃくちゃ使っています。
毎日の事務作業、ほぼ全部 Cowork に任せていて。
カレンダーへの予定登録から、ファイル整理、メールの下書きまで、本当に優秀な部下が隣にいる感覚なんですよ。

しかもこれが、月額 3,000 円ですよ？
コスパが良すぎて、正直怖いくらいです。

Web 開発でも、同じように使えます。

「〇〇のアプリを作って、ブラウザで動作確認して、おかしな点があれば修正して、問題なければデプロイして」という一連の流れを、全部 Cowork に任せられます。

デプロイというのは、作ったアプリをサーバーに乗せて、インターネット上で誰でもアクセスできる状態にすることです。
通常この作業は手順が多くて時間がかかるんですが、Cowork に「デプロイまで全部やって」と言えば、その流れを丸ごと自動でこなしてくれます。

作る、確認する、直す、デプロイする。
この繰り返しが、全部 Cowork の中で完結するんです。

神機能 3 つ目は、Skills です。

Web サイトを作るたびに長い指示を打ち込んで、少し時間が経つと「いつもどう指示していたっけ？」と分からなくなって、出力がブレてしまう...。
そんな悩みを、この機能が解決してくれます。

Skills は、指示書（プロンプト）＋プログラム＋テンプレートを、まとめて 1 つのパッケージにしたものです。

使い方はシンプルで、チャットに「/スキル名」と打つだけです。
「/web-artifacts-builder」のように入力すると、登録した指示がそのまま再現されて、毎回同じ精度で出力が返ってきます。

しかも Anthropic が公式に用意したスキルが、ディレクトリに揃っているので、自分で作らなくてもすぐに使い始められます。

チャット画面だけでなく、Claude Code、先ほど解説した Cowork でも使うことができます。
今回はチャット機能での例をお見せします。

デスクトップアプリもしくは Web 版の Claude を開いて
サイドバーのカスタマイズをクリック
スキルをクリック

これが Skills の管理画面になります。

では、公式に用意されたスキルをインストールしましょう。

その場合は、+をクリックして、
スキルを閲覧をクリック

このように公式で用意されているスキル一覧が表示されるので、
web-artifacts-builder をクリックして、インストールしましょう。
このスキルは、ランディングページから本格的な Web アプリまで、見た目も動きも揃った完成品を一発で出力してくれるスキルです。

インストールが完了したら、チャットに「/」と打ってみてください。
インストールしたスキルが一覧で表示されるので、「/web-artifacts-builder」を選択します。

そのまま、こう指示してみます。

アーティファクトというのは、Claude が生成したコードや成果物を、チャット画面の横にリアルタイムでプレビュー表示してくれる機能のことです。
完成したアプリをその場で確認しながら作業できるので、開発のテンポが一気に上がります。

通常の Claude で同じことをやろうとすると、「こういう機能をつけて」「このデザインで」「この構成で」と、細かく指定しながら何度もやり取りしなければいけません。
でも、スキルを使えば、最初の一言だけでここまでの精度が出ます。

僕自身も独自の Skill を作って使ってるんですが、もうこれなしには戻れないですね。
一度体験すると、分かります。この感覚。
