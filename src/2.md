(スライドを表示)

まずは Claude Design から。

Claude Design は、ブラウザ上で使える、スライド作成や Web サイト・Web アプリのデザイン作成に特化した AI ツールです。

Claude Code と同じように、会話を通じて Web サイトを作れるんですけど、Claude Design の面白いところは、会話している画面とまったく同じ画面で、リアルタイムにサイトの見た目を確認しながら作れるところなんですよね。

しかも、さらに良くするための専用ツールが、画面上にそのまま用意されているので、「なんか違う」と思ったら、その場で即座に調整できる。

さらに、出来上がったものを、そのまま Claude Code に持っていけるので、デザインから実装まで、一気通貫で進められるのが本当に強いんです。

これを初めて使った時、「やばすぎだろ、、、」って思いました。

ただ、Claude Design 単体で Web サイトを作ると、どうしても少しだけ "AI で作りました感" が残ってしまうんですよね。

(スライドを表示)

そこで登場するのが、GPT Image 2。

これは ChatGPT の中で動く、最新の画像生成モデルです。

前のバージョンから大幅に進化していて、まず画像全体の "AI っぽさ" が一気に減りました。

以前は、どこか不自然で、見た瞬間「あ、AI が作ったやつだ」って一発でわかる仕上がりになりがちでした。

それが gpt image 2 になって、人が作ったと感じさせるような、自然な仕上がりの画像が普通に出せるようになったんです。

さらに、以前は画像の中に日本語を入れようとすると、文字化けしてしまって、正直使い物にならなかったんですよね。

それが今は、誤字や文字化けがほぼ発生しないレベルまで改善されていて、Web サイトのデザインモックとしても十分使えるクオリティになったんです。

実は僕、以前は画像生成に ChatGPT をあまり使っていなくて、Gemini の Nano banana をメインで使っていたんですよ。

ただ最近、試しに ChatGPT で画像を生成してみたら、「え、めっちゃ良い画像出すじゃん!」って驚いて。

それ以来、Web サイトのデザイン画像は、完全に gpt image 2 に切り替えました。

「DENSITY」では、余白の広さを、ぎゅっと詰めるか、ゆったり取るかを、3 段階で調整できます。

「SHAPE」では、カードの角の丸みを、シャープ・ソフト・ピル型から選べます。

たったこれだけで、サイト全体の印象が、丸ごとガラッと変わるんですよ。
