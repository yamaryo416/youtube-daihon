# 参考動画 1: 良いポイント

- URL: https://youtu.be/dvk3rm24mrk?si=nC5cXJ0aSWzlingO
- タイトル: 【激震】外注なら30万のWebサイト、Claude Design × GPT image 2で作ってみた！

---

## 1. 冒頭の惹きが非常に強い

「外注なら30万のWebサイトを2〜3時間で作った」という具体的な数字とビフォーアフターのギャップを冒頭で提示している。さらに完成済みLPの画面を最初に見せることで、視聴者に「自分もこれを作れるようになる」というゴールイメージを最短で植え付けている。タイトル・サムネ・OPで提示する「価値の予感」が一致しているため離脱が起きにくい構成。

## 2. 独自ワークフローという「希少性」を明確に打ち出している

「SNSで誰もこの方法をやってない、僕が自分で考えた」と明言することで、他の解説動画との差別化を演出。AI 系の解説動画は「ツール紹介」止まりが多い中、ツールを **組み合わせる発想** にフォーカスしている点が情報価値として高く、ファンが付きやすい設計。

## 3. 「素材がデザインの質を決める」という核心の言語化

ただツールの使い方を説明するのではなく、「なぜ Claude Design 単体だと AI 感が抜けないのか」「なぜ素材が必要なのか」を、現役フロントエンドエンジニアの一般的なワークフロー（デザイナーが透過素材を用意 → FE が CSS で配置）と対比しながら言語化している。**理由の説明が明確** で、視聴者が手法の本質を理解できる。

## 4. 工程をステップで分けて見せている

- ① モック画像生成 → ② 背景素材抽出 → ③ ロゴ・アイコン抽出 → ④ Claude で素材分割・透過化 → ⑤ Claude Design に流し込み → ⑥ Claude Code に移行 → ⑦ セクション追加とブラッシュアップ

という流れが画面の切り替えとともに自然に展開されており、視聴者が真似しやすい。手順がブラックボックスになっていない。

## 5. 失敗・試行錯誤もそのまま見せている

「1発で成功したわけではなくて、正方形のグリッドで出力し直してもらった」「ピクセル単位を合わせないとプログラム処理がうまくいかなかった」という **詰まりポイントとその解決法** を隠さず見せている。視聴者が真似した時に同じ壁にぶつかっても、動画を見返せば突破できる実用性がある。

## 6. プロンプト設計の具体例を提示している

「目的・業種・ターゲット・トンマナ・デザイン要件」という構造化プロンプトの型を画面で見せ、「分からなかったらAIに聞けばいい」という補足までセットで提供。**初学者でもすぐ真似できる** 落とし所になっている。

## 7. Claude Design → Claude Code への移行手順を実演

Share → Hand to Claude Code でコピーして、ローカルの Claude Code に貼り付けるだけで移行できる、という操作を画面で実演。**「Claude Design は週次制限があるから途中で Claude Code に逃がす」** という実運用上のテクニックも添えており、実際に使う人の悩みを先回りして解消している。

## 8. `/frontend-design` スキルへの言及

セクション追加時に「スラッシュ・フロントエンドデザイン」スキルを使うと AI 感が抜けると紹介。**自分のチャンネルの他コンテンツ（スキル活用）への内部リンク** にもなっており、視聴者を次の動画に誘導する導線として機能している。

## 9. デプロイまで完結させている

「これ既にデプロイしてあるので、概要欄から誰でも見れます」と告知。**実物を触れる** ことで動画の信憑性が圧倒的に上がり、概要欄クリックの動機にもなっている。

## 10. 締めの抽象化が深い

「ベストプラクティスは確立されないので自分で構築する」「Claude Code スキルも自分の業務に合わせて育てる必要がある」という、**手法を超えた思想・心構え** で締めている。視聴者に「この人をフォローし続ければ最先端の考え方が得られる」という印象を残せている。

## 11. 「2、3時間で作った」の根拠が映像で担保されている

冒頭の主張が、動画内で実際に手順を踏むことで裏付けられている。一連のフローを実演で見せているので、「言ってるだけ」感がなく信頼性が高い。

## 12. レスポンシブ対応・ハンバーガーメニューまで見せている

「PCで完成」で終わらず、スマホ表示も最後にチェック。**実務でそのまま使えるレベル** であることを示しており、視聴者の「結局PCだけでしょ?」という疑念を先回りして潰している。
